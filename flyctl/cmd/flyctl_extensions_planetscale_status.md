Show status of a PlanetScale database


## Usage
~~~
flyctl extensions planetscale status <name> [flags]
~~~

## Options

~~~
  -h, --help   help for status
~~~

## Global Options

~~~
  -t, --access-token string   Fly API Access Token
      --verbose               Verbose output
~~~
