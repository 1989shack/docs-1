Run a performance test against a URL


## Usage
~~~
flyctl curl <URL> [flags]
~~~

## Options

~~~
  -h, --help   help for curl
  -j, --json   JSON output
~~~

## Global Options

~~~
  -t, --access-token string   Fly API Access Token
      --debug                 Print additional logs and traces
      --verbose               Verbose output
~~~
