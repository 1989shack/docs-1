Permanently destroy a database

## Usage
~~~
flyctl extensions planetscale destroy <name> [flags]
~~~

## Options

~~~
  -h, --help   help for destroy
  -y, --yes    Accept all confirmations
~~~

## Global Options

~~~
  -t, --access-token string   Fly API Access Token
      --verbose               Verbose output
~~~
