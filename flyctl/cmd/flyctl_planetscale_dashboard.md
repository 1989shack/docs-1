Visit the PlanetScale database dashboard

## Usage
~~~
flyctl planetscale dashboard <database_name> [flags]
~~~

## Options

~~~
  -h, --help   help for dashboard
~~~

## Global Options

~~~
  -t, --access-token string   Fly API Access Token
      --verbose               Verbose output
~~~
